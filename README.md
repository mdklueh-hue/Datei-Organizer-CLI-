# CLI Datei Organizer

Das ist ein Datei Organizer, den ich mit Python gebaut habe. 
Das Programm sortiert automatisch Dateien in einem ausgewählten Ordner 
nach Typ, zum Beispiel Bilder, Dokumente, Videos oder Musik.

## Funktionen

- Ordnerpfad eingeben
- Dateien automatisch nach Dateiendung erkennen
- Passende Unterordner automatisch erstellen
- Dateien in die richtigen Ordner verschieben
- Unbekannte Dateitypen werden in "Sonstiges" verschoben

## Verwendete Technologien

- Python 3
- os (für Arbeiten mit Dateien und Ordnern)
- shutil (zum Verschieben von Dateien)

## Was ich mit dem Projekt gelernt habe

- Arbeiten mit dem Dateisystem
- Nutzung von os.listdir() und os.path
- Dateiendungen erkennen
- Schleifen und Bedingungen sinnvoll einsetzen
- Strukturierter Aufbau eines CLI-Programms

## Projekt starten

1. Datei main.py oder file_organizer.py ausführen:
   
   python main.py

2. Ordnerpfad eingeben (z.B. Downloads)

Die Dateien werden dann automatisch sortiert.
